# AI-question 项目说明

本项目为“AI-question”课设，包含前端 Vue3、后端 FastAPI、AI接口等模块。**目前前后端已基本联通，部分业务流程已打通。**

---
## 快速启动

1. 进入 `vue/` 目录，安装依赖并启动前端
   ```sh
   npm install
   npm run dev```
2. 进入 fastapi/ 目录，配置数据库并启动后端
   - 参考 fastapi/readme.md 配置环境及数据库
3. 浏览器访问前端页面`http://localhost:5173/`，体验完整功能
---

## 项目结构

- `vue/`：前端主目录，基于 Vue3 + Vite，包含页面、组件、路由、API 封装等。
- `fastapi/`：基于 FastAPI 的后端接口，含数据库模型、接口实现、文件上传等。
- `BESTTTTT_AI_Q/`：AI接口，提供题目生成、答案解析、文件转文字等功能。
- `README.md`：主项目说明文档。

---

## 主要功能进度

### 1. 前端（vue/）

- **已实现：**
  - 用户注册、登录、登出
  - 讲座创建、文件上传
  - 讲座列表、历史数据、个人中心
  - 听众端：加入演讲、实时PPT同步、答题、解析、热力图统计
  - 演讲者端：讲座控制、发题、PPT同步、数据统计
  - API 封装（`api.js`），所有接口均支持表单方式调用
  - 路由守卫、页面跳转、状态管理
- **使用方法：**
  1. 进入 `vue/` 目录，执行 `npm install`
  2. 运行 `npm run dev` 启动本地开发服务器
  3. 浏览器访问页面，体验完整业务流程

### 2. 后端（fastapi/）

- **已实现接口：**
  - `/sign` 用户注册
  - `/login` 用户登录
  - `/upload` 文件上传
  - `/download` 文件下载
  - `/speaker/lecture_create` 创建讲座

- **使用方法：**
  1. 进入 `fastapi/` 目录，按 `readme.md` 配置环境
  2. 启动后端服务（如 `python file.py`）
  3. 确保数据库、依赖、端口配置正确

### 3. AI接口（BESTTTTT_AI_Q/）

- 支持题目生成、答案解析、文件转文本等（需按需对接）
